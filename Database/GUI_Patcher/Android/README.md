# Android / Kivy Prototype

This is the first Android GUI prototype.

Goal:
- Keep the existing Tkinter GUI for desktop.
- Add a lightweight Kivy Android UI.
- Reuse the existing patching backend where possible.

Current status:
- UI prototype only.
- Android patching still needs Android-compatible `ndstool` and `blz` binaries.
- Desktop Kivy testing can be done before APK packaging.
